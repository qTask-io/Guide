Task Overview: All tasks are displayed on the main screen. Use the round button to record a task using your voice. The app will convert your voice into text and add it to the task list.

Record a Task: Press and hold the round button to record a voice task. You don’t need to be perfect – just speak naturally as if talking to someone. Do not release the button during recording.

Create an Ideal Task: The app creates tasks perfectly but allows you to edit or add more details by pressing the round button and following the previous steps.

Task Sharing: Each task has a message icon on the right. Tap it to generate a ready-made request to ask someone to complete the task. Share it easily via messenger or email.
